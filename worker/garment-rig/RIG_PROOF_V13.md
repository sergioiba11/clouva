# CLOUVA AutoRig fresh-run proof

Every accepted avatar rerig must report:

- Worker version `v12-real-blender-autorig`.
- Profile version `clouva-blender-autorig-v12-official-reference`.
- A unique run ID and positive Blender duration.
- SHA-256 hashes for input and output, which must differ.
- The original clean Meshy avatar as input.
- Official Unreal reference rig, validated hand fitting, and at least 98.5% weighted vertices.

Creator Studio must reject and avoid persisting any result that does not satisfy this contract.
